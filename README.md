# moshu-backend

管理团队页面

获取成员信息： http://127.0.0.1:8000/team_manage/get_member/
任命管理员： http://127.0.0.1:8000/team_manage/appoint/
删除成员： http://127.0.0.1:8000/team_manage/delete/
撤销管理员： http://127.0.0.1:8000/team_manage/revoke/

### 登录注册
