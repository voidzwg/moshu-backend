## 一、管理团队页面

### 1、获取成员信息

**说明：**获取成员信息

**传参**：团队id

**请求方式：**post

**返回**：此团队的所有成员的所有信息

### 2、任命管理员

**说明：**更改该成员的身份为管理员

**传参**：成员id、团队id

**请求方式：**post

**返回**：

​	{

​	error：成功为0，否则为其它

​	msg：对应提示信息

​	}

### 3、删除成员

**说明：**删除成员

**传参：**成员id、团队id

**请求方式：**post

**返回**：

​	{

​	error：成功为0，否则为其它

​	msg：对应提示信息

​	}

### 4、撤销管理员

**说明：**更改该管理员身份为普通成员

**传参：**用户的id、团队id

**请求方式：**post

**返回**：

​	{

​	error：成功为0，否则为其它

​	msg：对应提示信息

​	}

## 二、邀请成员页面

### 1、获取所有成员
